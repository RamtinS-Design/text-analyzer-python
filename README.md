# Text Analyzer (Python)
A simple Python script that analyzes a sentence or paragraph and returns useful text statistics.

## Features
- Count the number of words
- Count the number of characters
- Find the most frequent word
It uses Python's built-in features and the collections.Counter module.

## How It Works
- words → counts the number of words
- characters → counts the number of characters
- frequent → finds the most frequently used word

## Example
Enter the word or the sentence: hello world hello 
Choose (words / characters / frequent): frequent 
Most frequent word: hello
